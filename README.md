### 1px解决方案

#### 使用方式
通过npm下载  or 直接拷贝1px.css
```
 npm i border-1px -S
```
引入
```
 @import 1px.less
 // or
 <link rel="stylesheet" href="./node_modules/border-1px/1px.css">
```
可用类名
  - `b-1px-l` 左边框
  - `b-1px-r` 右边框
  - `b-1px-t` 上边框
  - `b-1px-b` 下边框
  - `b-1px-tb` 上下边框
  - `b-1px` 全边框

代码量十分的少,所以不必担心造成冗余问题
